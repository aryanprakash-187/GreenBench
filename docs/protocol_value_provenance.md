# Protocol value provenance

## DNeasy Blood & Tissue
- Tissue protocol uses 180 µL Buffer ATL.
- Then 20 µL Proteinase K.
- Then 200 µL Buffer AL and 200 µL ethanol.
- Then 500 µL Buffer AW1, 500 µL Buffer AW2, and 200 µL Buffer AE.

## GeneJET Genomic DNA Purification Kit
- Cultured mammalian cell protocol gives exact volumes for Lysis Solution, Proteinase K Solution, RNase A Solution, 50% ethanol, Wash Buffer I, Wash Buffer II, and Elution Buffer.

## MagJET Genomic DNA Kit
- Manual cultured mammalian cell protocol gives exact volumes for NaCl, Digestion Solution, Proteinase K Solution, RNase A Solution, Lysis Buffer, isopropanol, magnetic beads suspension, Wash Buffer 1, Wash Buffer 2, and Elution Buffer.

## Q5 Hot Start High-Fidelity 2X Master Mix
- 50 µL reaction composition is explicit.
- Template DNA is variable, so the seeded quantitative sheet excludes biological template volume from shared-prep math.

## Platinum II Hot-Start PCR Master Mix (2X)
- 50 µL reaction composition is explicit.
- Routine thermal-cycle table is explicit.
- No separate final extension is listed in the routine example.

## JumpStart REDTaq ReadyMix Reaction Mix
- 50 µL reaction uses 25 µL ReadyMix with primers + template + water.
- Primer stock is 20 µM and final concentration is 0.4 µM, so 1 µL per primer is used in the seeded quantitative sheet.

## Agencourt AMPure XP PCR Purification
- AMPure XP volume = 1.8 × sample volume.
- 96-well cleanup uses 200 µL 70% ethanol twice and 40 µL elution buffer.

## MagJET NGS Cleanup and Size Selection Kit
- Cleanup tables specify 5 µL beads, 700 µL Binding Mix, 400 µL Wash Solution twice, and 50 µL Elution Buffer for a 100 µL DNA sample.

## AxyPrep Mag PCR Clean-up
- AxyPrep uses 1.8 × sample volume bead ratio, 200 µL 70% ethanol twice, and 40 µL elution buffer.
- For the seeded bundle, a 50 µL PCR cleanup default is used, giving 90 µL bead reagent.
