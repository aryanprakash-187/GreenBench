BenchGreen final seed CSV bundle (9 seeded protocols)

What is included:
- protocols_selected.csv
- reagent_term_map.csv
- equipment_term_map.csv
- consumables_term_map.csv
- protocol_reagents.csv
- protocol_thermal_profiles.csv
- reagent_stability.csv
- waste_rules_map.csv
- overlap_rules.csv
- equipment.csv
- operators.csv

Important modeling assumptions used to keep joins clean:
1) protocol_reagents.csv uses exact protocol_name values from protocols_selected.csv
   and exact reagent_raw_term values from reagent_term_map.csv.
2) Optional PCR additives (GC Enhancer, mineral oil, betaine, DMSO, TE alternative
   elution) are included with zero default volume so foreign-key joins succeed without
   forcing every run to use them.
3) template DNA is modeled at 1 uL default where vendor manuals leave the volume variable.
4) MagJET NGS cleanup uses the manual cleanup workflow (100 uL sample, 5 uL beads,
   700 uL Binding Mix, 400 uL wash x2, 50 uL elution). Binding Buffer and isopropanol
   are decomposed from the vendor's fresh Binding Mix table.
5) This bundle is engineered to match the existing seed reagent map. If you later decide to
   track Proteinase K or additional ethanol rows explicitly, revise reagent_term_map.csv first
   and then add corresponding protocol_reagents rows.
